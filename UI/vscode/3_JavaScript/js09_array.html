배열이란?

배열의 선언

array객체의 메소드

indexOf(): 배열에서 요소가 위치한 인덱스를 리턴

concat(배열명): 두 개 또는 세 개의 배열을 결합

join(): 배열을 결합하여 문자열로 반환

reverse(): 배열의 순서를 뒤집는다.


push(): 배열의 맨 뒤에 요소 추가, pop(): 배열의 맨 뒤에 요소 제거


shift(): 배열의 맨 앞에 요소 제거, unshift(): 배열의 맨 앞에 요소 추가


slice(): 배열의 요소를 선택해서 잘라내기, splice(): 배열의 index위치의 요소 제거 및 추가


toString(): 배열을 문자열로 반환
