인코딩과 디코딩

escape : 영문 알파벳, 숫자, 일부 특수문자(@, *, -, _, ., /)를 제외한 모든 문자를 인코딩
encodeURI : escape에서 인터넷 주소에 사용되는 일부 특수문자(:, ;, /, =, ?, &)는 변환하지 않는다
encodeURIComponent : 알파벳과 숫자를 제외한 모든 문자를 인코딩

unescape : escape로 인코딩된 문자를 디코딩
decodeURI : encodeURI로 인코딩된 문자를 디코딩
decodeURIComponent : encodeURIComponent로 인코딩된 문자를 디코딩

infinity와 NaN

자바스크립트는 0으로 숫자를 나누면 infinity라는 값이 들어간다.
자바스크립트가 표현할 수 없는 숫자는 NaN(Not a Number)로 표시.
